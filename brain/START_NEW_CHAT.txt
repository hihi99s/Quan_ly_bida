🎯 IMPORTANT CONVENTIONS:

Database:
- H2 in-memory (auto-seed via DataSeeder.java)
- 27 PriceRules seeded automatically
- Enums: TableType (POOL, CAROM, VIP), DayType (WEEKDAY, WEEKEND, HOLIDAY)

Billing:
- 3 price bands per table type per day: 8-12, 12-17, 17-23
- Always check if PriceRule exists before calculate
- totalAmount includes tableCharge + serviceCharge - discount

Session Management:
- Status: ACTIVE, PAUSED, COMPLETED
- Table Status: AVAILABLE, PLAYING, PAUSED, RESERVED, MAINTENANCE
- pauseStart tracks when pause begins

Security:
- CSRF enabled on /admin/** (tokens in forms)
- CSRF disabled on /api/** and /ws/** (stateless)
- Roles: ADMIN, STAFF

WebSocket:
- Endpoint: /ws
- Topic: /topic/tables
- Broadcast every 5 seconds via @Scheduled
